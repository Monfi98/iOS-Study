# 고급 연산자

## 오버 플로우(Over Flow)

- C언어나 Objective-C의 산술연산자에서는 오버플로우를 허락했지만, 스위프트에서는 오버플로우를 기본적으로 허락하지 않는다.
- 특정한 경우에, 특정 패턴을 구현하기 위해 오버플로우를 허용하는 경우가 필요하기 때문에 스위프트에서 오버플로우 연산자를 구현해 놓았다.
    - `&+` : 오버플로우 더하기 연산자
    - `&-` : 오버플로우 빼기 연산자
    - `&*` : 오버플로우 곱하기 연산자

```swift
var b = UInt8.min
b = b &- 1
```
